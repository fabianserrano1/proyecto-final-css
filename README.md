# Proyecto Final CSS

Este proyecto forma parte de la **tarea entregable del módulo de CSS**.

El objetivo del proyecto es completar una página web desarrollada previamente en HTML y añadirle estilos utilizando CSS siguiendo buenas prácticas de desarrollo web.

---

# Descripción del proyecto

El proyecto consiste en la creación de una página web completa y responsive simulando un sitio web real para un cliente.

Se ha trabajado principalmente en la **maquetación, estructura semántica del HTML y estilos con CSS/SASS**, aplicando un enfoque **mobile first** para garantizar una correcta visualización en diferentes dispositivos.

---

# Objetivos del proyecto

- Aplicar estilos a una estructura HTML existente
- Utilizar buenas prácticas de maquetación
- Implementar un diseño **responsive**
- Organizar los estilos utilizando **SASS**
- Mejorar la estructura visual y la experiencia de usuario
- Aplicar estados interactivos como **hover en botones y enlaces**

---

# Tecnologías utilizadas

Las tecnologías utilizadas en este proyecto han sido:

- **HTML5** → estructura semántica del contenido
- **CSS3** → estilos y diseño visual
- **SASS / SCSS** → organización modular de estilos
- **Vite** → herramienta de desarrollo y build
- **Git** → control de versiones
- **GitHub** → alojamiento del repositorio
- **GitHub Pages** → publicación del proyecto

---

# Características principales

El proyecto incluye las siguientes características:

- Diseño **responsive**
- Metodología **Mobile First**
- Uso de **media queries**
- Organización de estilos con **SCSS**
- Uso de **iconos SVG**
- Estructura HTML semántica
- Estados **hover en elementos interactivos**
- Navegación entre distintas páginas

# Instalación del proyecto

Si se quiere ejecutar el proyecto en local:

Clonar el repositorio:

```

git clone https://github.com/TUUSUARIO/proyecto-final-css.git

```

Entrar en la carpeta del proyecto:

```

cd proyecto-final-css

```

Instalar dependencias:

```

npm install

```

Ejecutar entorno de desarrollo:

```

npm run dev

```

Generar versión final del proyecto:

```

npm run build

```
